Página Pessoal – Projeto Prático Web1 (ADS)

Descrição:
Este projeto consiste em uma página pessoal desenvolvida como atividade prática em sala de aula da disciplina Web1 (ADS). O objetivo é treinar HTML e CSS, criando um site responsivo, interativo e acessível, aplicando conceitos de layout, cores, tipografia e boas práticas de web design.
Esta atividade é não avaliativa, servindo apenas como prática prática de sala de aula.

Tecnologias Utilizadas:
-HTML5 – Estrutura semântica e conteúdo da página;
-CSS3 – Estilização moderna, responsividade, cores e efeitos visuais.
Futuramente será incluído JavaScript para mais interatividade

Funcionalidades Implementadas:
-Layout responsivo, com blocos de informações lado a lado
-magens e cards interativos com hover e zoom
-Botões e links animados com hover e transformações
-Fade-in em seções ao rolar a página
-Formulário com validação básica e campos de senha
-Links para redes sociais (Instagram, Facebook, GitHub)
-Scroll suave e acessibilidade para leitura confortável

Observações:
Este projeto é um treino prático em sala de aula, destinado a praticar conceitos de web design e acessibilidade. O site pode ser incrementado futuramente com JavaScript para funcionalidades adicionais.
